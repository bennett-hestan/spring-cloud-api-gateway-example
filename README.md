# Spring Cloud API Gateway Example

An example of API gateway with Spring Cloud.


## Components

- `server`
  - A simple REST server
  - Spring Boot Web
  - Sleuth
- `api-gateway`
  - A couple of REST server and client
  - Spring Boot Web
  - Feign
  - Hystrix
  - Sleuth
  - Zuul Proxy
- `client`
  - A simple REST client
  - Feign
  - Hystrix
  - Sleuth
